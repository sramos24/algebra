Concept

What does "simplify a radical" mean?

A radical like √48 is simplified when there's no perfect square hiding inside it. The trick is to factor out all perfect squares.

Perfect squares to memorize: 1, 4, 9, 16, 25, 36, 49, 64, 81, 100, 121, 144...

For variables: every pair of the same variable comes out of the radical as one copy. Odd powers leave one behind inside.

√(x⁶) = x³    √(x⁵) = x²√x    √(x⁸) = x⁴
Worked example — Problem 1

Simplify 3√(24x³)

1
Factor the number into perfect square × leftover: 24 = 4 · 6
2
Factor the variable into a perfect square × leftover: x³ = x² · x
3
Take the square root of each perfect square part: √4 = 2 and √x² = x
4
Multiply everything that came out by the original coefficient (3):
3 · √(4 · 6 · x² · x) = 3 · 2 · x · √(6x) = 6x√(6x) ✓
Worked example — Problem 2

Simplify √(32a⁶b³)

32 = 16 · 2 → √16 = 4 a⁶ = (a³)² → √(a⁶) = a³ b³ = b² · b → √(b²) = b, one b stays inside = 4a³b · √(2b) ✓
Quick check: whatever is left inside the √ should have no perfect square factors, and no variable with an exponent ≥ 2.
Concept

Adding and subtracting radicals

Radicals behave exactly like variables — you can only add or subtract them if the part under the √ is identical.

Think of √2 like a variable "w".
3√2 + 5√2 = 8√2   just like   3w + 5w = 8w.

Always simplify each radical first, then look for matching terms.

Worked example — Problem 4

Simplify −2√32 + √72 + 4√2

1
Simplify each: √32 = 4√2, √72 = 6√2
2
Rewrite: −2(4√2) + 6√2 + 4√2
3
Combine: −8√2 + 6√2 + 4√2 = 2√2 ✓
Worked example — Problem 5

Multiply (√3 + 2)(−2√3 + 4) using FOIL

Key fact: √a · √a = a. The radical completely disappears!
F: √3 · (−2√3) = −2 · 3 = −6 O: √3 · 4 = 4√3 I: 2 · (−2√3) = −4√3 L: 2 · 4 = 8 = −6 + 4√3 − 4√3 + 8 = −6 + 0 + 8 = 2 ✓
The middle terms cancelled out perfectly — that happens a lot with conjugate pairs.
Concept

Why rationalize the denominator?

In math, leaving a radical in the denominator (bottom of a fraction) is considered unfinished. We multiply by a clever form of 1 to remove it without changing the value.

Worked example — Problem 3

Simplify √50 / (2√27)

1
Simplify numerator and denominator first: √50 = 5√2, 2√27 = 6√3
2
Multiply by √3/√3 to clear the denominator:
5√2 · √3 = 5√6 6√3 √3 6 · 3 = 5√6 / 18 ✓
Worked example — Problem 6

Simplify 2 / (3 − √2) — use the conjugate

When there's a binomial in the denominator (two terms, one with a radical), multiply by the conjugate — same terms but the middle sign flips.

Conjugate of (3 − √2) is (3 + √2).
This uses: (a − b)(a + b) = a² − b² → the radical disappears!
2 · (3 + √2) = 2(3 + √2) (3 − √2) (3 + √2) 9 − 2 = 6 + 2√2 ✓ 7
Concept

Three steps for every radical equation

1
Isolate — get the radical alone on one side
2
Square both sides — this removes the √
3
Check every answer — squaring can create fake answers called extraneous solutions
An extraneous solution is one that makes the original equation false when you plug it back in. Always check!
Worked example — Problem 7

Solve 3 + 2√(5x + 1) = 11

Step 1 — subtract 3: 2√(5x+1) = 8 Step 1b — divide by 2: √(5x+1) = 4 Step 2 — square both sides: 5x+1 = 16 Step 3 — solve: 5x = 15 → x = 3 Check: 3 + 2√(5·3+1) = 3 + 2√16 = 3 + 8 = 11 ✓
Worked example — Problem 8

Solve √(x + 2) = 2x + 1

This one gives a quadratic after squaring — two solutions to check!

Square both sides: x + 2 = (2x+1)² = 4x² + 4x + 1 Rearrange: 0 = 4x² + 3x − 1 Factor: (4x − 1)(x + 1) = 0 Solutions: x = 1/4 or x = −1
Must check both!
Check x = 1/4: √(1/4 + 2) = √(9/4) = 3/2 2(1/4) + 1 = 3/2 ✓ Valid! Check x = −1: √(−1 + 2) = √1 = 1 2(−1) + 1 = −1 ✗ Extraneous! Answer: x = 1/4 only
Concept

The distance formula

d = √[ (x₂ − x₁)² + (y₂ − y₁)² ]

This is the Pythagorean theorem in disguise. The horizontal gap (x₂−x₁) and vertical gap (y₂−y₁) are the two legs. The distance d is the hypotenuse.

Leave your answer as an exact radical — do not round! The test says "leave exact."
Worked example — Problem 9

Distance between (−3, 0) and (−5, 3)

1
Label the points: x₁ = −3, y₁ = 0, x₂ = −5, y₂ = 3
2
Find the differences: x₂ − x₁ = −5 − (−3) = −2 and y₂ − y₁ = 3 − 0 = 3
3
Plug in: d = √((-2)² + 3²) = √(4 + 9) = √13 ✓
Worked example — Problem 10

Find missing coordinate: (n, −6) and (−1, 2), d = 10

10 = √[(−1 − n)² + (2 − (−6))²] 10 = √[(−1 − n)² + 64] Square both sides: 100 = (−1 − n)² + 64 36 = (−1 − n)² ±6 = −1 − n Case 1: +6 = −1 − n → n = −7 Case 2: −6 = −1 − n → n = 5 Answer: n = −7 or n = 5 ✓
Concept

Pythagorean theorem

a² + b² = c²
where c is always the longest side (the hypotenuse)

Use it to: find a missing side, check if a triangle is a right triangle, or set up a word problem equation.

Worked examples — Problems 12 & 13

Is it a right triangle?

Sides 6, 8, 10

6² + 8² vs 10² 36 + 64 vs 100 100 = 100 ✓ Yes!

Sides 10, 24, 26

10² + 24² vs 26² 100 + 576 vs 676 676 = 676 ✓ Yes!
Always put the largest number as c (the hypotenuse). If you mix it up you'll get the wrong answer.
Worked example — Problem 11

Word problem: one leg is 4 longer, hypotenuse = √58

1
Name the legs: shorter = x, longer = x + 4
2
Set up equation: x² + (x+4)² = (√58)² = 58
3
Expand: x² + x² + 8x + 16 = 58
4
Simplify: 2x² + 8x − 42 = 0, divide by 2: x² + 4x − 21 = 0
5
Factor: (x + 7)(x − 3) = 0 → x = 3 (reject x = −7, can't have negative length)
6
Legs are 3 and 7. Check: 3² + 7² = 9 + 49 = 58 = (√58)² ✓
Concept

Graphing y = √(x − h) + k

Start with the parent function y = √x, then shift it using h and k.

h → moves the graph LEFT or RIGHT (starting x-value)
k → moves the graph UP or DOWN (starting y-value)

Domain: x ≥ h     Range: y ≥ k
Sign trick: (x − h) shifts RIGHT. (x + 2) would actually shift LEFT 2, because h = −2.
